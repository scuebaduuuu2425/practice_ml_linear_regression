# House Price Predictor using Linear Regression 
<h3>🤦‍♀️Author By - Aswasana</h3>
<br>

## Project Overview
This project is a Machine Learning regression model built using the California Housing dataset. The model predicts house prices based on various housing features such as median income, house age, population, latitude, and longitude. 

<br>

## Objective
The objective of this project is to understand the complete Machine Learning workflow:
- Data Loading
- Data Exploration
- Data Visualization
- Data Preprocessing
- Model Training
- Model Evaluation
- Prediction

<br>

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Joblib
- Jupyter Notebook

<br>

## Dataset

California Housing Dataset from Scikit-learn.

<br>


## Model Used
Linear Regression
Linear Regression Formula:

y = b0 + b1x1 + b2x2 + ... + bnxn

<br>

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

<br>

## Features
- Data Visualization
- Correlation Heatmap
- Actual vs Predicted Plot
- Model Saving using Joblib
- Small Prediction UI Script
- Model Evaluation

<br>
